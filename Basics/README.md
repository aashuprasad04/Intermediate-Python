## 1. dictionary<br>
- Dictionary is a built-in-data type that stores data in key-value pairs.
- Syntax:
  ```py
  dic = {key1: value1, key2: value2}
  ```
- Dictionary are
  - unordered
  - mutable
  - indexed by keys, which means you cna change their content after creation, and each key must be unique.
