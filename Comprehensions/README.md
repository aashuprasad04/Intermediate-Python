# Comprehension :<br>
Comprehension is a short and simple way to create a new collection (like list, set, or dictionary) using a single line of code.<br>
it is mainly used istead of loopos.

## 1. List Comprehension
#### Syntax :
```py
[expression for item in iterable if condition]
```
